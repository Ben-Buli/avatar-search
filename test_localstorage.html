📊 localStorage 功能說明

  • 自動儲存：第一次載入時自動將 JSON 資料設定到 localStorage
  • 雙重儲存：修改後同時更新 localStorage 和伺服器檔案
  • 一鍵複製：提供完整的 JSON 資料一鍵複製功能
  • 本地管理：可以查看、清除、匯出、匯入 localStorage 資料
