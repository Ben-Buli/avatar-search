🐛 問題原因

在 styles.css 中有兩個不同的 .download-btn 樣式定義：

  • 第 503 行：用於圖片卡片的懸停下載按鈕（opacity: 0）
  • 第 3499 行：用於頂部導航的下載按鈕

這造成了樣式衝突，導致頂部導航的下載按鈕被隱藏。

🛠️ 解決方案

將圖片卡片的樣式改為更具體的選擇器：

/* 修改前 */ .download-btn { opacity: 0; /* ... */ } /* 修改後 */ .image-card .download-btn { opacity: 0; /* ... */ }

這樣可以避免樣式衝突，確保頂部導航的下載按鈕正常顯示。
